This div is using a background color defined in the variable --theme-success. It uses a feature in CSS that is not supported in any version of IE. This following code is used to define a fallback to IE 11 when required. 
.inline-color {
    /* defaults for IE 11 */ 
    color:white;
    background-color:green; 
    /* modern browsers support css vars */
    background-color:var(--theme-success);
    color:var(--theme-light);
}
